# XX实验室 · 项目与代码平台

欢迎来到 **XX实验室 GitHub Organization**。  
本组织用于集中管理实验室课程实验、科研项目与技术 Demo。

---

## 📌 使用说明（请所有成员阅读）

- 每位同学 **仅维护一个或多个属于自己的仓库**
- ❌ 禁止修改、提交他人仓库代码
- ✅ 每个项目必须包含：
  - `README.md`（项目说明）
  - 可运行代码或 Demo
- 项目成果请通过 **Release / Demo 文件 / Pages** 方式展示

## 🧑‍💻 项目规范（摘要）

每个项目仓库建议包含以下结构：

```text
project-name/
├── README.md        # 项目说明（必需）
├── src/             # 源代码
├── demo/            # 演示文件（视频 / 图片 / 可执行文件）
└── docs/            # 相关文档（可选）
